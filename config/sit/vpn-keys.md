只加密 private_key 字段
1. 原始 vpn-keys.yaml
yaml
keys:
  - name: master-1
    private_key: <master_private_key>
    public_key: <master_public_key>
2. 使用 ansible-vault encrypt_string 加密 private_key
bash
ansible-vault encrypt_string 'my-secret-private-key' --name 'private_key'
示例输出（加密后是 YAML 结构）：
